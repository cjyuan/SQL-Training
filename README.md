# SQL-Training

Codewar Kata -- SQL types: create a type
https://www.codewars.com/kata/6618f11abcd4e4445d998373/train/sql


## Steps
1. Define the Composite Type
1. Create the Canonical Constructor (Normalization + Reduction)
1. Implement GCD Helper Function
1. Add Accessor Functions (numerator, denominator)
1. Implement Arithmetic Functions (+, -, unary -, *, /, %)
1. Register Arithmetic Operators
1. Implement Comparison Function
1. Register Comparison Operators (=, <>, <, >, <=, >=)
1. Define Casts To/From integer, bigint, numeric, text/varchar
1. Support Mixed-Type Operators (rational ↔️ bigint)
1. Enforce Canonical Form Invariants (reduced, positive denominator, 0 → 0/1)
